Install path

Admin App Distribution

1. Host the PWA

Put this folder on HTTPS through Cloudflare Pages, Netlify, Vercel, or similar static hosting.

2. Install on iPhone

Open the hosted link in Safari, tap Share, then Add to Home Screen.

3. Install on Desktop

Open the hosted link in Chrome or Edge and choose Install app.

4. Connect shared data

Create the Supabase table from the included SQL file, then save the settings above.
